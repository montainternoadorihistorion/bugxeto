# 添付文書群を踏まえての検討内容

## 1. この文書の目的

この文書は、以下の資料群と、その後の追加検討を踏まえて、IJK日本招致、とくに**大本・天恩郷で開催する場合の進め方、資金計画、支出順序、リスク管理**を整理したものです。

- `IJK日本開催地を大本の天恩郷に絞った調査・検討2.md`
- `IJK日本開催地を大本の天恩郷に絞った調査・検討3.md`
- `IJK日本開催地を大本の天恩郷に絞った調査・検討4.md`
- `IJK大本亀岡開催_予算調査プロンプト_2027-2030.md`
- `IJK日本開催の予算推定と会場比較（IJK2028予算比較＿ChatGPT DeepResearch_ベスト）.pdf`

結論から言うと、大本・天恩郷でのIJK開催は、**条件付きで現実的**です。

ただし、その条件はかなり明確です。

1. 大本側の強い現物支援があること。
2. 参加費は全額前払いで、未払い申込者を開催判断人数に含めないこと。
3. 24週価格表を最終価格として扱い、TEJO会員割引をさらに重ねないこと。
4. 若者個人が立替・借金・個人契約をしないこと。
5. 遠足、記念品、奨学金、渡航補助を本体会計に混ぜすぎないこと。
6. 入金されたお金をすぐ使わず、人数ゲートごとに支出を解放すること。

大きな方針としては、**「若者主導・大本後見・会計独立」**が最もわかりやすく、リスクも低いと考えます。

## 2. 現時点の前提

### 2.1 参加費は全額前払い

参加登録は、**参加費全額の入金をもって確定**するのがよいです。

30%前払い、最低デポジット、残額後払いなどは、参加者にも会計担当にも分かりにくく、未収金管理も残ります。日本開催では、準備費・中止判断・若者保護を明確にするため、**申込だけでは枠を確保せず、全額支払済み人数だけを開催判断に使う**方針が適切です。

### 2.3 TEJO会員割引は重ねない前提

近年のIJKでは、TEJO会員割引が設定されることがあります。しかし、日本案では、24週価格表そのものを**宿泊・食事・会場費等を含む最終価格**として設計しているため、ここからさらに一律のTEJO会員割引を重ねると、収支がかなり悪化します。

有料280人の場合、仮に€50割引を追加すると、減収は概ね次のようになります。

| TEJO会員比率 | 減収 |
|---:|---:|
| 30% | 約67万円 |
| 50% | 約112万円 |
| 70% | 約157万円 |

したがって、TEJOに対しては、**この24週価格表は追加割引なしの最終支払額である**と説明するのが望ましいです。どうしても割引が必要な場合は、本体会計ではなく、TEJO側基金・指定寄付・奨学金会計で補填するのが安全です。

## 3. 収入見込み

### 3.1 Group構成の仮定

ここでは、現時点で比較的あり得る想定として、**中国・東アジア多め型**を置きます。

| Group | 比率 |
|---|---:|
| A | 25% |
| B | 45% |
| C | 20% |
| Ĉ | 10% |

この構成で24週価格表に均等に登録が入る場合、平均単価は約**€286/有料参加者**です。

### 3.2 24週を3区分に分けた平均単価

24週を8週ずつに分けると、平均単価は次のようになります。

| 期間 | 平均単価 |
|---|---:|
| 第1〜8週 | 約€238/人 |
| 第9〜16週 | 約€286/人 |
| 第17〜24週 | 約€334/人 |
| 24週均等 | 約€286/人 |

第1〜8週は安いので、早く売れるほど平均単価は下がります。ただし、早く人数が見える価値は大きいです。主催側にとっては、**早く登録してもらうことによる安心感**の方が重要です。

### 3.3 有料参加者280人の場合の収入

有料参加者280人、€1=160円で見ると、収入は次の通りです。

| 登録の入り方 | 登録分布 | 収入 |
|---|---|---:|
| 前倒し型 | 1〜8週 50%、9〜16週 30%、17〜24週 20% | 約1,217万円 |
| 均等型 | 各週均等 | 約1,281万円 |
| 後ろ倒し型 | 1〜8週 20%、9〜16週 30%、17〜24週 50% | 約1,346万円 |

このため、有料280人モデルでは、参加費収入は概ね**1,200万〜1,350万円**と見るのが自然です。

### 3.4 有料300人・350人の場合の収入

| 有料参加者数 | 前倒し型 | 均等型 | 後ろ倒し型 |
|---:|---:|---:|---:|
| 300人 | 約1,304万円 | 約1,373万円 | 約1,442万円 |
| 350人 | 約1,521万円 | 約1,602万円 | 約1,682万円 |

つまり、支出が1,300万円前後なら有料280〜300人でかなり現実的です。

一方、支出が1,550万〜1,600万円級になるなら、**有料340〜350人規模**が必要になります。

## 4. 支出見込み

### 4.1 支出総額の大枠

添付資料群には複数の試算がありますが、現時点では次のように読むのがわかりやすいです。

| シナリオ | 支出目安 | 意味 |
|---|---:|---|
| 超強支援 | 約930万円 | 大本の現物支援が非常に強い |
| 強支援・実務的 | 約1,050万〜1,250万円 | 本命レンジ |
| 上振れ管理値 | 約1,300万〜1,425万円 | 遠足・記念品まである程度含めた上限感 |
| 上振れ＋10%予備費 | 約1,568万円 | かなり安全側の総額 |
| 標準支援 | 約1,470万〜1,670万円 | 参加費だけでは不足しやすい |
| 限定支援 | 約1,900万〜2,300万円 | 参加費だけでは困難 |
| 保守・悪化 | 約2,500万〜2,800万円 | 招致判断として危険域 |

ここで重要なのは、**1,300万円前後まではまだ管理可能だが、1,500万円を超えると参加費だけではかなり苦しくなる**という点です。

### 4.2 費目別の上振れ値

これまでの検討で置いた費目別の上振れ値は次の通りです。

| 費目 | 上振れ値 | コメント |
|---|---:|---|
| 宿泊・寝具・清掃 | 300万円 | 大本の強支援があれば妥当。外部宿泊なら不可能 |
| 食事・飲料 | 400万円 | 20食前後。大本厨房協力が前提 |
| 光熱・冷房・水道 | 70万円 | 8月開催なので削りすぎ注意 |
| 衛生・熱中症対策 | 80万円 | 氷、飲料、塩分、救急、静養室 |
| 会場・設営 | 80万円 | 会場費減免が前提 |
| 音響・Wi-Fi・投影 | 150万円 | 既存設備が使えれば下がる |
| 印刷・名札・受付備品 | 50万円 | 妥当 |
| 決済・送金・返金手数料 | 65万円 | 収入の3〜5%程度 |
| 下見・事前会議 | 50万円 | 若者無償なら低く抑えられる |
| プログラム | 50万円 | 低コスト企画中心 |
| 遠足 | 100万円 | 本体に入れると重い。希望者別料金が安全 |
| 記念品 | 30万円 | 余剰金がある場合のみ |
| 奨学金・渡航補助 | 別会計 | 本体予算に混ぜない |

この上振れ値を足すと、次のようになります。

| 範囲 | 合計 |
|---|---:|
| 本体必須部分のみ | 約1,295万円 |
| 遠足込み | 約1,395万円 |
| 遠足＋記念品込み | 約1,425万円 |
| 遠足＋記念品＋10%予備費 | 約1,568万円 |

この整理から見ると、**本体支出は1,300万円以内を目標**にするのがよいです。

遠足や記念品まで本体に含めると1,400万円台になります。さらに10%予備費まで見れば1,568万円です。

### 4.3 1,568万円を参加費で集めるには何人必要か

1,568万円を参加費だけで集める場合、€1=160円では必要なユーロ収入は約**€98,000**です。

東アジア多め型で見ると、必要な有料参加者数は次の通りです。

| 登録の入り方 | 平均単価 | 必要な有料参加者 |
|---|---:|---:|
| 前倒し型 | 約€272/人 | 約361人 |
| 均等型 | 約€286/人 | 約343人 |
| 後ろ倒し型 | 約€300/人 | 約327人 |

つまり、1,568万円を参加費だけで集めるなら、**有料参加者327〜361人**が必要です。

運営委員20人が参加費免除なら、総参加者数では概ね**350〜380人規模**になります。

したがって、1,568万円は「無理な数字」ではありませんが、**280人モデルではなく350人モデルの予算**として扱うべきです。

## 5. 予算上の管理目標

実務的には、次のように見るのがわかりやすいです。

| 本体支出 | 判断 |
|---:|---|
| 1,000万〜1,200万円 | 強支援が効いており、かなり良い |
| 1,200万〜1,300万円 | 有料280〜300人で概ね現実的 |
| 1,300万〜1,425万円 | 遠足・記念品まで入る上振れ領域。注意しながら可能 |
| 1,500万円超 | 参加費だけでは危険。追加支援が必要 |
| 1,600万円超 | 標準支援モデルに落ちている可能性が高い |
| 2,000万円超 | 限定支援・外部調達型。価格表維持は困難 |

短く言うと、

> 本体予算は1,300万円以内を目標。  
> 遠足込みなら1,400万円前後。  
> 1,500万円を超えるなら警戒。  
> 1,600万円を超えるなら追加支援なしでは危険。

という整理です。

## 6. 販売開始前からの宣伝戦略

### 6.1 宣伝は販売開始の2〜3カ月前から始める

24週価格表は、販売開始後に毎週価格が上がります。

したがって、販売開始と同時に初めて周知するのでは遅いです。

理想は、**販売開始の2〜3カ月前から宣伝を始める**ことです。

販売前にやるべきことは次の通りです。

- 公式案内ページを出す
- 参加希望フォームを開く
- TEJO・各国青年エスペランチストに周知する
- 中国・韓国・台湾・東南アジア向けの案内を準備する
- 24週価格表を見せて、早期購入の動機を作る
- 大本開催の意味を説明する
- 「第1週が最安値」であることを繰り返し伝える

ここは、若者の無償作業でかなり進められます。

### 6.2 販売開始時期

実務上は、次の時間軸が安全です。

| 時期 | 内容 |
|---|---|
| 開催40〜36週前 | 宣伝開始、関心登録開始 |
| 開催32〜30週前 | 24週価格表で販売開始 |
| 開催24〜22週前 | 第8週ゲート |
| 開催16〜14週前 | 第16週ゲート |
| 開催8〜6週前 | 第24週締切 |
| 開催6〜2週前 | 食材・寝具・備品・当日運営を本発注 |
| 開催2週前〜当日 | 小口・緊急支出のみ |

もし「開催24週前に販売開始」とする場合、24週価格表の最終週が開催直前になりすぎます。その場合は、第17〜24週を実質的な**late registration**扱いにし、大きな発注判断は第16〜20週までに済ませる必要があります。

## 7. 入金後の支出順序

### 7.1 基本方針

入金されたお金は、**入金順にそのまま使う**のではなく、用途別に分けて管理するべきです。

特に重要なのは、次の考え方です。

> 参加費は、入金された瞬間から全額使えるお金ではない。  
> 返金・中止・人数不足の可能性がある間は、一部を預り金としてロックする。

つまり、参加費は次の順番で扱います。

1. 決済・返金・送金手数料を見込む。
2. 返金・中止用の残高を先に確保する。
3. 第8週、第16週、第24週の人数ゲートで支出可能額を広げる。
4. 人数確定後に生活基盤へ本格支出する。
5. 遠足・記念品・奨学金は後回しまたは別会計にする。

### 7.2 支出優先順位

| 優先 | 支出 | 方針 |
|---:|---|---|
| 0 | 若者の無償作業 | まず現金を使わず進める |
| 1 | 決済・返金・送金手数料 | 必ず発生する実費 |
| 2 | 返金・中止用の残高 | 先に確保して触らない |
| 3 | 大本との条件確定 | 宿泊、食事、会場、光熱、清掃の単価確認 |
| 4 | 生活基盤 | 食材、寝具、衛生、給水、熱中症対策 |
| 5 | 最低限の当日運営 | Wi-Fi、音響、受付、印刷、表示 |
| 6 | プログラム | 低コスト企画を中心に |
| 7 | 遠足 | 希望者別料金が望ましい |
| 8 | 記念品・豪華演出 | 余剰金か寄付がある場合のみ |
| 9 | 奨学金・渡航補助 | 本体会計ではなく別会計 |

### 7.3 時期別の支出イメージ

| 時期 | 使ってよいお金 | 主な使い道 | 注意点 |
|---|---:|---|---|
| 宣伝開始〜販売前 | 0〜10万円、最大30万円程度 | 告知ページ、関心登録、翻訳、SNS、TEJO向け下準備 | 若者無償作業中心 |
| 第1〜8週 | 累計30〜80万円程度まで | 決済実費、最低限の案内、ビザ書類準備、軽い印刷、下見実費 | 入金の大半は使わない |
| 第8週ゲート | 大きな発注はまだしない | 支払済150〜180人に届くか確認 | 未達なら支出凍結 |
| 第9〜16週 | 累計150〜300万円程度まで | 大本条件確定、食事設計、Wi-Fi/音響見積、必要最低限の予約 | まだ贅沢支出はしない |
| 第16週ゲート | 本格発注の手前 | 200〜230人未満なら縮小案 | 230人以上で通常準備へ |
| 第17〜24週 | 原則は発注準備中心 | 最終募集、ビザ対応、備品確定 | 人数確定まで大口発注を抑える |
| 第24週締切後 | 本格支出開始 | 食材、寝具、清掃、給水、受付、表示、音響 | 支出の中心は生活基盤 |
| 開催2週間前〜当日 | 小口・緊急費中心 | 氷、飲料、救急、追加印刷、現地調整 | 当日精算を増やしすぎない |
| 終了後 | 精算 | 未払精算、返金、会計報告、余剰金処理 | 透明性が重要 |

## 8. 使ってはいけないお金の使い方

早期入金があると、主催側は安心して支出したくなります。しかし、第1〜8週の入金は、まだ「大会が確実に成立したお金」ではありません。

特に、早い段階で避けるべき支出は次の通りです。

- 貸切バスの大きな前払い
- 外部ケータリングの大口契約
- 大型音響・照明・AVの前払い
- 記念品の大量発注
- Tシャツやグッズの大量発注
- 返金不能な遠足予約
- 豪華な印刷物
- 人数未確定の食材発注

これらは、第24週締切後、または十分な人数と保証枠が見えた後に判断すべきです。

## 9. 会計体制

### 9.1 大本口座でもよい

口座は、大本口座を使う形でもよいと考えます。

大本開催であれば、外部から見た信頼性も高く、若者個人の口座を使わずに済むためです。

ただし、大本の通常会計と混ざると見えにくくなるため、次の条件は必要です。

| 条件 | 内容 |
|---|---|
| IJK専用補助簿 | 大本通常会計と分けて管理 |
| 入金者別台帳 | 誰が、いつ、いくら払ったかを管理 |
| 返金用残高の見える化 | 使ってよいお金と使ってはいけないお金を区別 |
| 二名承認 | 大本側1名＋若者側/会計担当1名など |
| 月次共有 | 若者実行委・大本側で残高と支出を確認 |
| 余剰金・赤字時の扱い | 事前に文書化 |

理想は、**大本名義口座＋IJK専用補助簿**です。

### 9.2 若者個人にリスクを背負わせない

若者が担うべきなのは、企画、広報、国際連絡、プログラム、参加者対応、現場運営です。

若者個人が担ってはいけないのは、次のものです。

- 個人口座での参加費受取
- 個人クレジットカードでの大口決済
- 返金不能な契約の個人名義締結
- 大会赤字の個人補填
- 後から精算される前提の大きな立替

若者主導で進めることと、若者に財務リスクを負わせることは別です。

## 10. 若者と大本関係者の役割分担

### 10.1 骨太な役割分担

| 担い手 | 主な役割 |
|---|---|
| 若いエスペランチスト | 広報、参加者募集、申込対応、SNS、プログラム、国際連絡、ボランティア運営 |
| 大本関係者 | 会場、宿泊、食事、施設ルール、地域調整、緊急時対応、現物支援 |
| 会計・リスク管理担当 | 入出金管理、返金、支出承認、予算表、赤字見通し、ゲート判断 |
| TEJO側 | IJK規程、Group分類、国際広報、公式性、参加者支援制度 |

大きく言えば、**若者が大会を作り、大本が生活基盤を支え、会計担当がリスクを守る**形です。

### 10.2 若者の無償作業で下げられる費用

若者が無償で頑張れるなら、以下はかなり現金支出を抑えられます。

- サイト制作
- 申込フォーム作成
- メール対応
- FAQ作成
- SNS運用
- 多言語案内
- 表計算会計
- 参加者管理
- 簡単なデザイン
- プログラム調整
- 当日受付

ただし、無償作業に頼りすぎると燃え尽きや属人化が起きます。作業を分担し、スプレッドシートや共有ドキュメントで引き継げる形にする必要があります。

## 11. Go/No-Goゲート

### 11.1 募集開始前

募集開始前に最低限そろえるべきものは次の通りです。

- 大本の支援条件の大枠
- 宿泊・食事・会場・光熱・清掃の考え方
- 大本口座または実行委員会口座の管理方法
- 返金ポリシー
- 支出承認ルール
- 第8週、第16週、第24週の判断基準
- TEJO追加割引をどう扱うか
- 大本側の赤字保証枠または再承認ルール

ここが曖昧なまま販売開始するのは危険です。

### 11.2 第8週ゲート

第8週は、最初の重要な判断点です。

| 指標 | 判断 |
|---|---|
| 支払済150〜180人以上 | 継続可能 |
| 150人未満 | 広報強化、支出凍結、縮小案検討 |
| 大本支援条件が未確定 | 大口発注禁止 |
| 外部発注が必要になっている | 原則保留 |

第8週時点では、入金があっても大口支出は避けます。

### 11.3 第16週ゲート

第16週は、開催可能性をかなり真剣に見る時点です。

| 指標 | 判断 |
|---|---|
| 支払済200〜230人以上 | 通常開催準備へ進む余地あり |
| 200人未満 | 縮小案を本格検討 |
| 標準支援止まり | 追加支援なしでは危険 |
| ビザ対応が遅れている | 中国・アジア参加者数を保守的に見る |

ここで、食事・寝具・Wi-Fi・音響などの見積を固めていきます。

### 11.4 第24週締切

第24週が実質的なGo/No-Go判断です。

| 支払済有料参加者数 | 判断 |
|---:|---|
| 230人未満 | 中止または大幅縮小 |
| 230〜250人 | 強支援と保証枠がなければ危険 |
| 250〜280人 | 強支援なら通常開催可能性あり |
| 280人以上 | 本命運営ライン |
| 340〜350人以上 | 1,500万円台予算も視野に入る |

第24週後に、食材、寝具、清掃、受付、表示、音響などを本格発注します。

## 12. 遠足・記念品・奨学金の扱い

### 12.1 遠足

遠足は、60万〜100万円超の費用になりやすいです。

本体参加費に含めると、収支をかなり圧迫します。

したがって、遠足は原則として次のどちらかが安全です。

1. 希望者別料金にする。
2. 助成金・寄付・スポンサーが取れた場合のみ本体に含める。

全員参加の豪華遠足を本体会計に入れるのは、かなり慎重に考えるべきです。

### 12.2 記念品

記念品は、余剰金または指定寄付がある場合だけでよいです。

最初から本体必須費に入れると、予算が硬くなります。

### 12.3 奨学金・渡航補助

奨学金・渡航補助は、本体会計に混ぜない方がよいです。

理由は、以下です。

- 本体収支が見えにくくなる。
- 赤字時に若者運営者へしわ寄せが出やすい。
- 寄付者に対する説明が難しくなる。
- TEJO向けにも、本体会計と参加支援会計を分けた方が説明しやすい。

したがって、奨学金・渡航補助は**別会計・指定寄付・TEJO系基金**で扱うのがよいです。

## 13. 保険・安全対策の考え方

保険は、最初から大きく見積もりすぎる必要はありません。

ただし、完全に無視するのも危険です。

現実的には、次の順番がよいです。

1. 大本の既存保険で、外部参加者の宿泊・食事・施設利用中の事故がどこまで対象になるか確認する。
2. 食中毒、熱中症、転倒、設備破損、行事中の事故について責任範囲を確認する。
3. 不足する部分だけ、最小限のイベント保険・賠償責任保険・旅行保険案内で補う。

保険を大きな予算項目にしすぎる必要はありませんが、**大本の既存保険・施設責任範囲の確認だけは必須**です。

## 14. 大本に確認すべき事項

大本に確認すべきことは、抽象的な「協力できますか」ではなく、費目別に具体化する必要があります。

| 優先 | 確認事項 | 望ましい回答 |
|---|---|---|
| 最優先 | 宿泊単価 | 研修実費に近い大幅値引き、または現物支援 |
| 最優先 | 食事単価 | 食材・厨房・配膳を実費中心で運用 |
| 最優先 | 会場費 | 大幅減免または現物支援 |
| 最優先 | 光熱・清掃 | 低額または一部大本負担 |
| 最優先 | 空調・給水 | 8月の大人数利用に耐える |
| 高 | Wi-Fi・電源 | 300人規模に不足しない |
| 高 | 深夜利用・音量 | IJK文化と施設ルールの両立 |
| 高 | 食事制限 | ベジ、ビーガン、ハラール、アレルギー対応 |
| 高 | 破損・事故時の責任 | 責任範囲を明文化 |
| 高 | 既存保険 | 外部参加者が対象になるか |
| 中 | 大本支援をTEJO資料に書けるか | 金額または現物支援として明記可 |
| 中 | 赤字保証枠 | 2〜3百万円以上あると安心 |

大本の強支援がどこまで文書化できるかが、この計画の核心です。

## 15. 何が起きると危険か

以下が起きると、予算は急に悪化します。

- 支払済参加者が230人未満にとどまる。
- 大本支援が強支援ではなく標準支援にとどまる。
- TEJO会員割引を追加で飲む。
- 遠足を本体負担にする。
- 外部ケータリング比率が上がる。
- Wi-Fi、音響、空調を大きく外注する。
- 返金・キャンセルが多い。
- 運営委員無料枠が増えすぎる。
- 円高に振れる。
- 物価高で食材費・光熱費が上がる。

特に大きいのは、**大本支援が弱まること**と、**参加者数が伸びないこと**です。

## 16. 最終的な太い方針

最終的な方針は、次のように言い切れます。

> 大本・天恩郷でのIJK開催は、24週価格表、全額前払い、若者の無償労務、大本の強い現物支援を組み合わせれば、かなり現実的になる。  
> ただし、入金された参加費をすぐ使うのではなく、返金・中止リスクを残し、第8週・第16週・第24週の人数ゲートごとに支出を解放する。  
> 本体予算は1,300万円以内を目標とし、遠足込みなら1,400万円前後、1,500万円超なら追加支援が必要、1,600万円超なら標準支援モデルに落ちている可能性が高い。  
> 若者は大会づくりを主導し、大本は場所と生活基盤を支え、会計はIJK専用に見える化する。若者個人に立替・借金・赤字責任を負わせない。

この形なら、若者側にも、大本側にも、TEJO側にも説明しやすいです。
